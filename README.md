# Das Wort des Tages
Get the word of the day from https://www.duden.de/wort-des-tages

## Example
```bash
-------------------------------------------------
Wort des Tages: quo vadis?
Bedeutung:      wohin wird das führen?; wer weiß
-------------------------------------------------

```
